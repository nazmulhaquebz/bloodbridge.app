**Version:** 1.0.0  
**Last Updated:** July 2026  
**Website:** https://bloodbridge-two.vercel.app

---

## Android Installation

### Requirements
- Android 8.0 (Oreo) or higher
- At least 50 MB free storage

### Steps

1. **Download** the APK from:  
   https://github.com/nazmulhaquebz/bloodbridge.app/releases/latest/download/bloodbridge.apk

2. **Enable Unknown Sources** on your device:
   - Android 8+: Settings → Apps → Special app access → Install unknown apps → [Your Browser] → Allow
   - Older Android: Settings → Security → Unknown Sources → Enable

3. **Install** the APK:
   - Open your Downloads folder
   - Tap `bloodbridge.apk`
   - Tap Install → Allow → Done

4. **Launch** Blood Bridge from your app drawer

---

## Windows Installation

### Requirements
- Windows 10 or higher (64-bit)
- At least 200 MB free storage

### Steps

1. **Download** the installer from:  
   https://github.com/nazmulhaquebz/bloodbridge.app/releases/latest/download/bloodbridge-setup.exe

2. **Run** `bloodbridge-setup.exe`

3. If **Windows SmartScreen** appears:
   - Click "More info"
   - Click "Run anyway"

4. **Follow the Setup Wizard:**
   - Accept the license agreement
   - Choose installation folder
   - Click Install

5. **Launch** Blood Bridge from Desktop shortcut or Start Menu

After installing on any platform:

1. **Register** as Donor or Requester
2. **Fill in** your blood type, city, and contact info
3. **Verify** your email with the OTP code we send
4. **Start using** Blood Bridge!

## Troubleshooting

| Problem | Solution |
|---------|---------|
| APK won't install | Enable "Install Unknown Apps" in Android settings |
| Windows blocks the EXE | Click "More info" → "Run anyway" in SmartScreen |
| App crashes on launch | Check your device meets the minimum requirements |
| Can't receive OTP email | Check your spam folder; whitelist `bloodbridge` |
| Login fails | Reset password via the "Forgot Password" link |
