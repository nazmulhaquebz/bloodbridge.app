# Blood Bridge — Release Notes

## v1.0.0 — Initial Public Release (July 2026)

### What's Included
- Full donor registration and matching system
- Real-time chat between donors and requesters (AES-256 encrypted)
- File and voice message sharing in chat
- Voice and video calling
- Blood request posting with urgency levels
- Admin dashboard for monitoring and moderation
- WhatsApp and phone contact integration
- Global donor search by blood type and city

### Platform Support
- Android 8.0+ (APK direct install)
- iOS 14+ (Web App / coming soon on App Store)
- Windows 10+ (Setup installer)
- macOS 12+ (coming soon)
