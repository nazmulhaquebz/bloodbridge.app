## 📖 Installation Guide

### 🤖 Android Installation (.apk)

> **Requirements:** Android 8.0 (Oreo) or higher

**Step 1 — Download**
- Tap the [📥 Download Android APK](https://github.com/nazmulhaquebz/bloodbridge.app/releases/latest/download/bloodbridge.apk) button above
- Wait for the `.apk` file to download to your phone

**Step 2 — Allow Unknown Sources**

Since Blood Bridge is distributed directly (not through Play Store), you need to allow installation from unknown sources:

- Go to **Settings → Security** (or **Settings → Privacy**)
- Enable **"Install Unknown Apps"** or **"Unknown Sources"**
- On newer Android: When prompted, tap **"Allow from this source"**

**Step 3 — Install**
- Open your **Downloads** folder
- Tap on `bloodbridge.apk`
- Tap **Install**
- Once installed, tap **Open**

**Step 4 — First Launch**
- Register as a **Donor** or **Requester**
- Verify your email with the OTP sent to you
- Start using Blood Bridge! 🩸

---

### 🪟 Windows Installation (.exe)

> **Requirements:** Windows 10 or higher (64-bit)

**Step 1 — Download**
- Click the [📥 Download Windows Installer](https://github.com/nazmulhaquebz/bloodbridge.app/releases/latest/download/bloodbridge-setup.exe) button above

**Step 2 — Run the Installer**
- Open your **Downloads** folder
- Double-click `bloodbridge-setup.exe`
- If Windows SmartScreen appears, click **"More info"** → **"Run anyway"**
  - *(This appears because the app isn't yet code-signed with a Microsoft certificate)*

**Step 3 — Follow the Setup Wizard**
- Choose an installation folder (default is recommended)
- Click **Next → Next → Install**
- Click **Finish** to launch Blood Bridge

**Step 4 — First Launch**
- A Blood Bridge shortcut will appear on your Desktop
- Open it and register your account to begin
